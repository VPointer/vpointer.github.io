变量有一定的命名规则：

  • 变量名只能包含字母、数字和下划线，且不能以数字开头
  • 变量名不能包含空格，一般用下划线分隔变量中的单词，也可以用驼峰命名法，但Python提倡用下划线
  • Python中的关键字和自带函数不能用于变量名
  • 变量名应该简短明了
  • 慎用小写字母 l 和大写字母O，因为这两个字母容易被看成数字1和0

同时也请注意，Python解释器不对代码进行拼写检查，应尽量避免命名错误，比如变量名中少写个字母之类的，否则会出现NameError。
