R语言-重抽样与自助法

发表于 2020-05-05 分类于 R语言 阅读次数： Disqus：

《R语言实战》笔记

许多实际情况中，统计假设并不一定被满足，比如数据抽样于未知或混合分布、样本量过小、存在离群点、基于理论分布设计合适的统计检验过于复杂且数学上难以处理等情况，这时基于随机化和重抽样的统计方法就可派上用场。

本篇探究两种应用广泛的依据随机化思想的统计方法：置换检验和自助法。

思维导图

置换检验Permutation test

置换检验，也称随机化检验或重随机化检验，数十年前就已经被提出，但直到高性能计算机的出现（因为要迭代多次），该方法才有了真正的应用价值。

研究表明，当样本含量较大时，置换检验得到的结果与经典的参数检验(t检验、F检验)近似；当样本含量较小时，置换检验要优于参数检验，并且其检验效能也高于秩和检验。

概念

置换检验的概念不好总结，以一个例子来说明。

假设有一个实验，它有两种处理方式，方式A和方式B，10个试验者被随机分配到这两组中，获得了一组得分：

现在想研究，这两种处理方式A和B的影响是否不同。

正态理论的处理方式

如果是参数方法，你可能会假设数据抽样自等方差的正态分布，然后用假设独立分组的双尾t检验来验证结果。此时，\(H_{0}\)为A处理的总体均值与B处理的总体均值相等，你根据数据计算了t统计量，将其与理论分布进行比较，如果t统计量落在你设定的置信区间外(比如95%)，那么你将拒绝\(H_{0}\)，并得出结论：在0.05的显著性水平下两组的总体均值不相等。

置换检验的处理方式

按置换检验的思路，如果A，B两种处理方式真的等价，那么上面这10个数据是可以随机分配个标签的（毕竟A和B等价嘛，谁是A谁是B有什么区别呢）。以这个假设为前提，按如下步骤进行检验：

  1. 与参数方法类似，计算观测数据的t统计量，称为\(t_{0}\)；
  2. 随机分配5个数据到A组，剩余的到B组；
  3. 重新计算t统计量；
  4. 重复步骤(2)~(3)，对每一种可能的随机分配计算t统计量，此处有252种可能(\(C_{10}^{5}\))的分配组合；
  5. 将252个t统计量按升序排列，得到基于(或以之为条件)样本数据的经验分布；
  6. 如果\(t_{0}\)落在经验分布中间95%部分之外，则在0.05的显著性水平下，拒绝两个处理组的总体均值相等的零假设。

这种计算了所有可能组合的统计量的置换检验叫做“精确”检验。当数据量增多时，“精确”检验将非常耗时，此时可以使用蒙特卡洛模拟，从所有可能的排列中进行抽样，获得一个近似的检验。

与参数方法的异同

置换方法和参数方法都计算了相同的t统计量。但置换方法并不是将统计量与理论分布进行比较，而是将其与置换观测数据后获得的经验分布进行比较，根据统计量值的极端性判断是否有足够的理由拒绝零假设。这种逻辑可以延伸至大部分经典统计检验和线性模型上来。

置换检验的优劣

  • 适用场景：置换检验真正发挥功用的地方是处理非正态数据(如分布偏倚很大)、存在离群点、样本很小或无法做参数检验等情况；
  • 置换检验主要用于生成检验零假设的p值，它有助于回答“效应是否存在”这样的问题，对于获取置信区间和估计测量精度比较困难。

用lmPerm包做置换检验

lmPerm包可做线性模型的置换检验。例如lmp()和aovp()函数即lm()和aov()函数的修改版。

lmp()和aovp()函数的参数与lm()和aov()函数类似，只额外添加了perm参数。

perm选项的可选值有Exact、Prob或SPR：

  • Exact根据所有可能的排列组合生成精确检验，若观测数大于10，perm="Exact"将自动默认转为perm="Prob"
  • Prob从 有可能的排列中不断抽样，直至估计的标准差在估计的p值0.1之下（具体值由Ca参数控制）
  • SPR使用贯序概率比检验来判断何时停止抽样

自助法

所谓自助法，即从初始样本重复随机替换抽样，生成一个或一系列待检验统计量的经验分布。 无需假设一个特定的理论分布，便可生成统计量的置信区间，并能检验统计假设(置信区间是否包含0)。

概念

依然以一个例子来说明。比如，样本现有10个观测，均值为40，标准差为5，现在你想计算一个样本均值95%的置信区间。

正态理论的处理方式

如果假设均值的样本分布为正态分布，那么\((1-\alpha / 2) \%\)的置信区间计算如下： \[ \bar{X}-t \frac{s}{\sqrt{n}}<\mu<\bar{X}+t \frac{s}{\sqrt{n}} \] 其中，\(t\)是自由度为\(n–1\)的\(t\)分布的\(1–α\)上界值。对于95%的置信区间，可得\(36.424<μ<43.577\)。

以这种方式创建的95%置信区间将会包含真实的总体均值。

自助法的处理方式

如果你确定样本是不是正态分布的，那么可以用自助法：

  1. 有放回地从样本中随机选择10个观测。有些观测可能会被选择多次，有些可能一直都不会被选中；
  2. 计算并记录样本均值；
  3. 重复\(n\)次步骤1～2；
  4. 将\(n\)个样本均值从小到大排序；
  5. 找出样本均值2.5%（位于\(0.025n\)位置的均值）和97.5%（位于\(0.975n\)位置的均值）的分位点，它们就是限定了95%的置信区间。

适用场景

如果你想估计样本中位数的置信区间，或者两样本中位数之差，该怎么做呢？正态理论没有现成的简单公式可套用，而自助法此时却是不错的选择。即使潜在分布未知， 或出现了离群点，或者样本量过小，再或者是没有可供选择的参数方法，自助法将是生成置信区间和做假设检验的一个利器。

boot包中的自助法

boot包扩展了自助法和重抽样的相关用途，可以对一个统计量(如中位数)或一个统计量向量(如一列回归系数)使用自助法。

一般来说，自助法有3个主要步骤：

  1. 写一个能返回待研究统计量值的函数：
    • 如果只有单个统计量(如中位数)，函数应该返回一个数值；
    • 如果有一列统计量(如一列回归系数)，函数应该返回一个向量；
  2. 为生成R中自助法所需的有效统计量重复数，使用boot()函数对上面所写的函数进行处理；
  3. 使用boot.ci()函数获取步骤2生成的统计量的置信区间。
